# utsuroi

Twitterのアイコンの色相をいじくるやつです。
テキトーに使ってね。謎に小洒落た名前をつけました。

# Requirement

* twitter 1.19.3
* requests-oauthlib 1.3.0

# Installation

```bash
pip install python-twitter
pip install requests requests-oauthlib
```
